# CSES Problem Set Solutions

Personal solutions to the [CSES Problem Set](https://cses.fi/problemset/), a curated collection of competitive programming problems.

## Structure

Solutions are organized by category, mirroring the CSES problem set sections:

```
introductory/
sorting-and-searching/
dynamic-programming/
graph-algorithms/
range-queries/
tree-algorithms/
mathematics/
string-algorithms/
geometry/
advanced-techniques/
```

## Usage

Each solution is self-contained. To compile and run a C++ solution:

```bash
g++ -O2 -o solution solution.cpp
./solution < input.txt
```

## Problem Set Categories

| Category | Problems |
|---|---|
| Introductory Problems | Warmup problems |
| Sorting and Searching | Classic sorting/searching challenges |
| Dynamic Programming | DP problems of varying difficulty |
| Graph Algorithms | BFS, DFS, shortest paths, flows |
| Range Queries | Segment trees, BITs, sparse tables |
| Tree Algorithms | Tree DP, LCA, centroid decomposition |
| Mathematics | Number theory, combinatorics |
| String Algorithms | Hashing, suffix arrays, automata |
| Geometry | Computational geometry |
| Advanced Techniques | Misc advanced topics |
