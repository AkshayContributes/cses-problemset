# Distinct Numbers

You are given a list of n integers, and your task is to calculate the number of distinct values in it.

## Input

The first input line has an integer n: the number of values.

The second input line has n integers $x_1, x_2, \ldots, x_n$.

## Output

Print one integer: the number of distinct values.

## Example

**Input:**
```
5
2 3 2 2 3
```

**Output:**
```
2
```

## Constraints

- $1 \leq n \leq 2 \cdot 10^5$
- $1 \leq x_i \leq 10^9$
