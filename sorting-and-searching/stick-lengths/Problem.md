# Stick Lengths

There are n sticks with various lengths. You want to make all sticks the same length. The cost of changing a stick's length from a to b is $|a - b|$. Find the minimum total cost.

## Input

The first line has an integer n: the number of sticks.

The second line has n integers $p_1, p_2, \ldots, p_n$: the lengths of the sticks.

## Output

Print one integer: the minimum total cost.

## Constraints

- $1 \leq n \leq 2 \times 10^5$
- $1 \leq p_i \leq 10^9$

## Example

**Input:**
```
5
2 3 1 5 2
```

**Output:**
```
5
```
